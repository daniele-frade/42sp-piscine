<br>

- n will be so that: `0 < n < 10`.

- If `n = 2`, here is the expected output:
```
>./a.out | cat -e
01, 02, 03, ..., 09, 12, ..., 79, 89$>
```

- Allowed functions:
```
write
```

- Prototype: 
```
void ft_print_combn(int n)
```
